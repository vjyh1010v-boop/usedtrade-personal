- 상품 삭제\_본인이 등록한 상품 삭제

- 카테고리별 조회\_전자기기, 의류, 도서, 생활용품 등 카테고리별 상품 조회

- 게시글 삭제\_작성자 본인만 게시글 삭제

- 게시글 카테고리\_자유게시판, 동네정보, 거래후기, 질문게시판 등 분류

- 게시글 좋아요\_마음에 드는 게시글에 좋아요

## 주윤희

- 로그인 기능
  - 맞지않습니다 등의 경고
  - 맞으면 다음화면 넘어가는 거
- 로그인 전 후로 보이는 홈 화면 같게
- 기본 홈화면 (게시판)
  - 상품 게시판 (상품목록보기)
    - 상품 등록
    - 상품 상세 페이지
    - 거래 상태 변경 - 상태업데이트 : 판매자가 버튼을 눌러 상품 상태 (판매중, 예약중, 판매완료) 변경
- ‘동네’ 기반 필터링 (간단한 검색)
  - 지역 설정 : 회원 가입 시 저장된 사용자가 지정한 ‘동네’ 정보를 활용
  - 우리 동네 보기 : 사용자의 지역과 일치하는 상품만 필터링하여 보여주는 기능
- 관심 상품 (찜 기능)
  - 관심 등록
  - 나의 관심 목록

### login.html 관련

2. 비밀번호 재설정

1. 검색 기능
2. 페이징
3. 즐겨찾기(찜)
